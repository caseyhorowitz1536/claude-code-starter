# Slash Commands

> Typed commands starting with `/` that trigger built-in actions or custom workflows inside a session.

## Why it matters
Slash commands are the keyboard shortcuts of Claude Code — fast, discoverable, and
unambiguous. They cover the housekeeping (login, memory, permissions) so you don't
have to phrase it as a request.

## How it works / how to use it
Type `/` to see the list. Common built-ins:

- `/init` — generate a starter [[CLAUDE.md and Memory|CLAUDE.md]].
- `/resume` — reopen an earlier [[Sessions and Resuming|session]].
- `/permissions` — review and edit allow/deny rules.
- `/memory` — view or edit memory files.
- `/help` — the full command reference.

Custom slash commands come from [[Skills]] and plugins — installing a plugin adds
its commands to your `/` menu.
