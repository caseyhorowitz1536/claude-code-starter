# Installing and Starting

> Install the CLI, log in, then run `claude` inside any project to begin.

## Why it matters
Getting installed and authenticated is the only gate between you and a working
agent. Once you're in, everything else is conversation.

## How it works / how to use it
Install on macOS/Linux with the official script:

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

Then start a session from a project directory:

```bash
claude
```

The first time, run `/login` to authenticate (Claude subscription or API key).
Useful first commands:

- `/help` — list everything you can do.
- `/init` — scan the project and generate a starter [[CLAUDE.md and Memory|CLAUDE.md]].

After that, just type what you want in plain English.
