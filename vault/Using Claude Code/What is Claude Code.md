# What is Claude Code

> Claude Code is an agentic coding assistant that lives in your terminal and works directly on your real codebase.

## Why it matters
Instead of copy-pasting snippets into a chat window, you let Claude read your
files, run commands, edit code, and verify the result — all with your approval.
It turns a chat model into a teammate that can actually do the work, then explain
what it did.

## How it works / how to use it
Run `claude` in any project directory to start a session. Claude works through
[[The Agentic Loop]]: it gathers context, takes actions (reads, edits, shell
commands), and verifies. The same agent is also available in your IDE (VS Code,
JetBrains), in the Claude desktop app, and on the web — the terminal is just the
most direct surface.

You stay in charge: every potentially destructive action asks first (see
[[Permissions and Settings]]), and you can interrupt or steer at any moment.
